# How to use

`npm run dev` 명령어로 개발 서버를 시작할 수 있고, `npm build`로 ES5 스펙으로 빌드할 수 있습니다. `npm start`를 치면 빌드 후 실행합니다.

# Error Codes

| Error Code    | description                                          |
| ------------- | ---------------------------------------------------- |
| NotFoundError | 액션을 찾을 수 없습니다. 404 스펙과 동일합니다.      |
| ServerError   | 서버에서 에러가 발생했습니다. 500 스펙과 동일합니다. |
